# audioWaveCanvas
这是一个录音实时录制绘制波形，最后转成整个波形的例子。
最近老是有人问我音频相关的一些东西，其实我也理解的很浅显。为了方便给位能够快速了解，给出一个，实时绘制波形的demo，录音完成加入整个完整波形的demo.
已经加入音频相似度 打分 比对
之后会加入变声 频谱绘制等功能
